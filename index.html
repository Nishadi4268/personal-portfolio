Hybrid AI-Based Fake News Detection System

An AI-powered fake news detection system that combines machine learning with semantic fact-verification. The model analyzes news articles, identifies key entities, checks facts against trusted knowledge sources like Wikipedia, and determines whether content is credible or misleading.

Key Features:
  • Hybrid detection pipeline: ML classifier + semantic fact validation
  • TF-IDF + Logistic Regression baseline model for classification
  • Named Entity Recognition (spaCy) to extract key persons/places/events
  • Semantic similarity with Wikipedia using MiniLM SentenceTransformer
  • Real-time predictions served via Flask REST API
  • Confidence scores & interpretable evidence (verified entity facts)
  • Dataset support for “True” & “Fake” articles
  • Model persistence & retraining using joblib

Tech Stack:
Python, scikit-learn, spaCy, HuggingFace SentenceTransformer (MiniLM), Pandas, Flask API, joblib

Future Enhancements:
Transformer-based classification (BERT/GPT), multilingual support, cloud deployment & MLOps pipeline
